# 概要
日々の学習計画を簡単に立てることができます。

# 機能紹介

## PWA化

ホーム画面に追加することで、PWA（サイトがネイティブアプリのような見た目になる）にすることができます。

これにより、ホーム画面からいつでも開くことができます。PC・モバイルどちらからでも利用可能です。（デザインはPC・モバイルともにモバイル版をベースにしています。）

さらに、データはブラウザ内のデータベース（IndexedDB）に保存されるので、ホーム画面から削除しない限り半永久的に利用できます。

## ログイン・バックアップ機能

画面下部の「ログイン」ボタンをクリックすると、GoogleアカウントまたはAppleアカウントでログインするよう要求されます。

利用したいアカウントを選択し、そのアカウントへのログイン操作を完了すれば、自動的にログインが完了します。

「教材一覧」（詳細は以下の通り）を一番下までスクロールすると、「アップロード」「ダウンロード」というボタンがあります。

「アップロード」をクリックすると、現在ブラウザに保存されているデータをアカウントに紐づけられたクラウドデータにバックアップされます。

「ダウンロード」をクリックすると、アカウントに紐づけられたクラウドデータを取得し、ブラウザ内のデータを書き換えます。

これらを使い分けることにより、複数端末でのデータのやり取りが可能になります。

## 教材一覧・予定リスト

まずアプリを開くと、「予定リスト」が表示されます。画面下部にある「表示切替」ボタンをクリックすると、「教材一覧」に切り替えることができます。

教材一覧では、一番下までスクロールした所にある「教材追加」ボタンをクリックすることで教材を追加できます。教科（英語、数学、現代文、etc…）を選び、教材名を入力して「OK」をクリックすれば登録完了です。

また、各教材のブロックをクリックすると、4つのボタンが出現します。

一番左から、「予定追加」「教材編集」「教材詳細メモ」「教材削除」となります。

このうち「予定追加」ボタンをクリックし、学習内容および開始時間を入力すると、予定を登録できます。（開始時間はなくても構いません。）

「表示切替」をクリックし、「予定リスト」に戻ると、追加した学習予定が表示されます。（開始時間が早い順に並びます。時間がないものは一番最後に表示されます。）

さらに、学習予定のブロックもクリックすることで利用可能な3つのボタンがあります。左から「完了（チェック）」「予定編集」「予定削除」となっています。

学習が完了したら、「チェック」ボタンをクリックしましょう！そうすると、その予定ブロックはリストの最後尾に移動し、次の予定が一番上に表示されるようになります。
