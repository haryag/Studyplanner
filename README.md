## 概要

毎日の学習予定を **シンプルに管理できるアプリ** です。

---

## 機能紹介

### ● PWAとして使える

* ホーム画面に追加すれば、アプリのように起動できます
* PC／スマホのどちらでも使用可能
  （見た目はスマホ向けレイアウト）
* データはブラウザの IndexedDB に保存
  → ホーム画面から削除しない限り **消えません**

---

### ● ログインとバックアップ

#### ログイン

* 画面下の「ログイン」ボタンから
* Google／Apple のどちらかでサインインできます

#### データの保存と読み込み

（※「教材一覧」画面の一番下）

| ボタン    | できること              |
| ------ | ------------------ |
| アップロード | 今のデータ → クラウドに保存    |
| ダウンロード | クラウドのデータ → この端末に反映 |

→ 端末を変えても **データを引き継げる**

---

## 画面と操作

### ● 画面の切り替え

* 起動時は **予定リスト** が表示
* 画面下の「表示切替」で
  「教材一覧」と行き来できます

---

### 教材一覧

* 一番下の「教材追加」で新しい教材を登録
  → 教科を選ぶ／名前を入力 → OK

* 教材をタップすると、次の4つが表示されます

| ボタン    | 内容         |
| ------ | ---------- |
| 予定追加   | この教材の予定を作る |
| 教材編集   | 名前の変更      |
| 教材詳細メモ | 補足メモを書く    |
| 教材削除   | この教材を消す    |

---

### 予定リスト

* 「予定追加」で
  **学習内容と開始時間** を入力して登録
  （時間は空欄でも問題ありません）

* 予定は **開始時間が早い順** に並びます
  ※時間なしの予定は最後に配置

予定をタップすると次の3つが表示されます

| ボタン      | 内容           |
| -------- | ------------ |
| 完了（チェック） | 完了扱いとして最後に移動 |
| 予定編集     | 内容変更         |
| 予定削除     | 削除           |

→ 完了を押すと「今やるべき予定」が先頭に来るようになります。
